Windows 客户端配置

 一、下载客户端

Windows客户端下载地址：https://github.com/shadowsocks/shadowsocks-windows/releases

我们推荐使用最新版的客户端，如果出现打开后直接显示程序未响应或停止工作等错误，请尝试历史版本，如3.3.2版本或者2.5.8的版本，
如果客户端程序运行有报错，3.3.5及其以上版本需要.net framework 4.6.2运行库，如果您未安装可以在这里下载：链接: http://pan.baidu.com/s/1i54qRDr 密码: xemi，
3.3.5以下版本请安装.net framework 4.0后再尝试运行！下载链接: 链接: http://pan.baidu.com/s/1bYbkpK 密码: rt55。

二、配置客户端

下载客户端并解压，运行Shadowsocks.exe，注：不可以在压缩包内运行程序，一定要解压出来，否则无法保存配置信息！

右键ss客户端，允许来自局域网的连接
 

弹出防火墙设置，全部勾选、允许访问


允许访问后，请勾掉允许局域网连接的选项即可。
请登录会员中心以后，打开https://my.yizhihongxing.com/clientarea.php?action=products，然后点击对应帐号后面的“查看详情/管理”进入到帐号的详情：


Hostname （Server IP）：服务器 IP 或 域名
Port ：你的服务器端口 （不同的账号的端口也都是不同的）
Password：你的密码
加密模式：一般为rc4-md5
 
下拉查看节点列表，选取其中一个节点后面的 二维码，右键ss客户端、服务器、扫描屏幕上的二维码
 

更改备注为节点服务器，以便于区分，确定
 

右键ss客户端,启用系统代理模式，打开浏览器如果能正常访问google，配置成功
 

注：如果当前版本可以正常使用，请关闭软件的自动更新（右键任务栏小飞机-更新-启动检测更新，把勾去掉）

Android 以及 iOS 客户端

安卓部分：

客户端下载链接: http://pan.baidu.com/s/1dEJbVzV 密码: 95j5

如果您是Android 6.x及其以上版本，请安装：链接: https://pan.baidu.com/s/1qYNnb6c 密码: ayk5
下载ss客户端并安装，下拉找到功能设置，选择绕过局域网及中国大陆地址。点左上角的“shadowsocks”英文进入新建节点配置页面

点击左下角“+”、根据实际情况选择配置新节点的方式，一般咱们选扫描二维码、打开pc网站节点信息页面，点选其中一个节点的二维码，扫描即可（没看见二维码图片，请下拉浏览器，图片处于页面正中间）
注：如果您只有手机在身边，那么请选择手动设置，对应官网后台节点信息填写相应数据
pc端节点信息页面：点首页会员登录=>用户管理中心=>我的ss帐号=>管理ss账号=>点击有效帐号后面的查看详情，即可看见节点信息


节点服务器配置好后，选中配置好的节点，返回，点击右上角的图标启动代理即可
注：如果弹出vpn连接提示，请勾选并确认
点击连接图标上面的“检查网络连接”可以测试节点是否可用，如果显示连接有效延迟多少ms，说明已经连接成功

ubuntu安装shadowsocks-qt5

打开终端执行：sudo add-apt-repository ppa:hzwhuang/ss-qt5，根据提示输入root用户密码，
待下图时，按enter继续执行


出现 用户名@ubuntu:~$ 即完成了命令
 

执命令：sudo apt-get update，这一步需要的时间稍微长一些，完成后
执命令：sudo apt-get install shadowsocks-qt5，根据提示按Y继续


三条命令执行完成安装也就完成了、到搜索页面搜索shadowsocks，查看您安装的程序，如果没有找到程序则未安装不完整，请重复上面的步骤


登陆官网后台，用户管理中心，我的ss帐号，管理ss账号，点击有效帐号后面的查看详情，即可看见节点信息，点击节点信息后面的二维码弹出图片
运行shadowsocks-qt5客户端，connection、add 、Scan QR Code using Capturer 、弹出扫描框


移动扫描框，完全覆盖二维码图片
 

扫描到节点信息，会自动填写完成，更改profile Name 为节点名字以便于区分、ok
 

选中扫描的节点，点击Connect 启用，启用成功后右侧 Disconnect 选项会亮起，此时客户端已经配置成功。Linux下需要配合浏览器插件才能代理，请转移到浏览器插件安装相应教程

基本配置说明

点击对应产品的“查看详情”将进入详细页面，我们将对产品的参数进行介绍：


流量限制：这里指的是您的套餐允许的最大流量。
加密方式：这非常重要，加密方式选择错误您将无法正常连接，我们采用rc4-md5的加密方式。
节点列表：在这里会列出您所有可以使用的服务器节点，您可以选择您觉得最快的线路。
连接端口：这非常重要，请一定不要输错！
连接密码：连接的密码，同样一定不要输错！
已经使用：这里指的是已经使用的流量，单位是MB。
剩余流量：这里指的是剩余可用的流量，单位是MB。
